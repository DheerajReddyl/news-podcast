Our next story is titled: Microsoft forced to make Windows 10 extended security updates truly free in Europe - The Verge. Microsoft will no longer require Windows Backup to enroll in its Extended Security Updates (ESU) Consumers will still need to enable Windows Backup or pay $30 for the year or redeem 1,000 Microsoft Reward points. The changes are only being made in the European Economic Area though, so everywhere else will still be required to turn on Windows Backup. Microsoft will still only get extended security updates for Windows 10 until October 13th, 2026, but businesses have the option to purchase up to three years of updates. And that concludes this report.
